# CEO Dashboard

A starter React + Vite dashboard for the CEO command center.

## Available Scripts

- `npm install` — install dependencies
- `npm run dev` — start the Vite development server
- `npm run build` — build the production bundle
- `npm run preview` — preview the production build

## Notes

- This project uses React, TypeScript, and Recharts.
- The dashboard currently includes a sample weekly revenue trend and revenue split chart.
- A native PHP API scaffold is provided under `public/api/` so the static frontend can be hosted on managed PHP hosting such as xneelo without requiring Node.js.
- Backend PHP endpoints support the existing form flows for enquiry submission, Home Connect leads, careers/applications, Telkom applications, and Supersonic fibre submissions.
